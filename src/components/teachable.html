评估指标

准确度 (Accuracy) = 正确预测总数 / 样本总数

精确度 (Precision) = TP / (TP + FP)（正类）

召回率 (Recall) = TP / (TP + FN)（正类）

F1 分数 = 2 * (Precision * Recall) / (Precision + Recall)（正类）
